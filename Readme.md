# CSApHLP — Tabu Search & Simulated Annealing

Solution to the **Capacitated Single Allocation p-Hub Location Problem (CSApHLP)**
implemented for **CW1 of CMSE11448 Soft Computing** at the University of Edinburgh.

> **Grade: 71/100 (Distinction)** — assessed by [Prof. Nader Azizi](https://www.business.ed.ac.uk/people/nader-azizi), UoE Business School

---

## Overview

This project implements and benchmarks two metaheuristic optimisation algorithms
for the CSApHLP:

- **Tabu Search (TS)** — neighbourhood search with a tabu list to escape local optima
- **Simulated Annealing (SA)** — probabilistic acceptance of worse solutions via a cooling schedule

Both methods are evaluated on standard benchmark datasets (CAB, TR, RGP) and compared
on solution quality and convergence behaviour.
