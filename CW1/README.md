A brief description of what this project does and its purpose.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Sarthak-Vishnu/B286266_code.git
   ```
2. Navigate to the project directory:
   ```
   cd CW1
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the application, execute the following command:
```
python src/main.py
```

To run the parameter tuning for Tabu Search, navigate to the `B286266_code\CW1` directory and execute:
```
python -m src.parameter_tuning
```

## Testing

To run the tests, navigate to the `B286266_code\CW1` directory and execute:
```
pytest tests/test_algorithms.py -v
```
